# Data_projects_TripleTen
This will show my portfolio and skills achieved. Projects I worked on while attending the Data Science TripleTen Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | InstaCart Shopping Analysis| This project analyzes customer shopping behavior using a modified version of the Instacart Online Grocery Shopping Dataset (2017 Kaggle competition). |
| 2 | Megaline Prepaid Plan Revenue Analysis | This project analyzes customer usage and revenue for Megaline, a telecom operator offering two prepaid plans: Surf and Ultimate. |
| 3 | Video Game Sales Analysis | This project analyzes historical video game sales data for Ice, an online game store that sells games worldwide. |
| 4 | Megaline Plan Recommendation Model | This project develops a machine learning classification model to recommend the most suitable mobile plan (Smart or Ultra) for Megaline customers based on their usage behavior. |
| 5 | Beta Bank Customer Churn Prediction | This project develops a machine learning model to predict customer churn for Beta Bank. |
| 6 | Oil Well Profit Prediction (OilyGiant) | This project develops a machine learning model to identify the most profitable region for new oil wells for the mining company OilyGiant. |
| 7 | Zyfra Gold Recovery Prediction | This project applies machine learning to optimize gold extraction for the industrial technology company Zyfra. |
| 8 | Insurance Data Analysis with Linear Algebra | This project explores how linear algebra and machine learning can be applied to solve business problems for the Sure Tomorrow insurance company. |
| 9 | Used Car Price Prediction | This project builds a machine learning model to estimate the market value of used cars for the Rusty Bargain car sales service. |
| 10 | Taxi Demand Forecasting (Time Series)| This project develops a time series forecasting model to predict the number of taxi orders for the next hour using historical airport taxi request data from Sweet Lift Taxi. |
